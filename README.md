# stepik-bankapp
![Jusan Singularity](https://ucarecdn.com/02b8ff49-8f2b-4ce9-be84-7d4bdc6b9b67/)

Это проект для вывода персонального счета 

## Возможности

Взаимодействовать с данным приложением можно через командную строку

- 1 - show accounts
- 2 - create account
- 7 - exit

## Запуск

    java Application
